# Steganon-1

<img width="2560" height="853" alt="Steganon-1-2" src="https://github.com/user-attachments/assets/29b55fec-c844-4033-a9dd-ddac52209aff" />

Steganon is an advanced, dual-node P2P encrypted communication terminal designed for secure data transmission in off-grid or urban environments where privacy is compromised. It is built around the high-performance Raspberry Pi RP2350 (Pico 2) microcontroller, the system leverages LoRa radio technology to facilitate decentralized, long-range text messaging and sensor data exchange without the need for cellular or internet infrastructure facilities.
