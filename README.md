----------------------------------------------

Steganon is a really advanced, dual-node P2P encrypted communication terminal designed for secure data transmission in off-grid or urban environments where privacy is compromised. It is built around the high-performance Raspberry Pi RP2350 (Pico 2) microcontroller, the system leverages LoRa radio technology to facilitate decentralized, long-range text messaging and sensor data exchange without the need for cellular or internet infrastructure facilities.

# BOM


##  Bill of Materials (BOM)

| Component | Function | Quantity |
| :--- | :--- | :---: |
| **Raspberry Pi Pico 2W** | Core Logic & Encryption (RP2350) | 2 |
| **RFM95W LoRa Module** | 866MHz Long-Range Communication | 2 |
| **NEO-M9N GNSS** | Location Tagging & Time-Sync | 2 |
| **1.3" I2C OLED** | User Interface & Message Display | 2 |
| **4x4 Matrix Keypad** | Manual Data & Cipher Entry | 2 |
| **TP4056 Module** | Li-Ion Battery Charging & Protection | 2|
| **MT3608 Converter** | 5V Boost Rail (GPS/OLED) | 2 |
| **AMS1117-3.3V** | 3.3V Logic Regulator (Pico/Radio) | 2 |
| **18650 Battery** | 3.7V Mobile Power Source | 4 |
| **SMA Antenna** | High-Gain 866MHz RF Interface | 2 |
